# Fitness-tracker

Python Fitness Tracker

Welcome to Personal Fitness Tracker, it's a simple but powerful Command Line Interface (CLI) application built in Python. Designed for quick access and weekly progress tracking, this programme helps you stay accountable and hit your fitness goals

## Features

- Log Everything:

Quickly record Exercise Type, Time, and Calories Burned for every day.

- Set Weekly Goals:

You can set targets for Calories burned and Workouts to be done each week.
 
- Progress report :

Get fast and simple bar chart of your weekly progress.

- Weekly Summary

See your weekly stats for the current Monday-Sunday week.

- Data Persistence

All your data is safely saved to a file.

## Problem Statement
Due to lack of tracking, people find it difficult to maintain routine in their exercise.
The goal of this project is to give users an easy-to-use, offline tool for tracking their workouts.

## Objectives

- Helps to track excercise daily
- Helps to record data
- Weekly display of excercise

## Getting Started

### Prerequisites

Python 3.x installed on your system.

Installation & Run

Save the Code: Checking that the provided Python code is saved as fitness_tracker.py.

Open Terminal: Locate the directory containing the file.

Execute: Run the programme using the Python interpreter.

### Usage Guide

When the programme runs. It will show a menu, just select the no. for the excercise you want to do...

Menu

Personal Fitness Tracker Menu:
- Workout History
- Setting Goal 
- View Weekly Summary 
- Checking Goals 
- Show Progress 
- Save and Exit
