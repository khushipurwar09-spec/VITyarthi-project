# Personal Fitness Tracker 

The Personal Fitness Tracker is a Python-based Command Line Interface application that supports recording workouts, tracking fitness goals, showing weekly progress, calculating BMI, and maintaining workout history.

It promotes accountability and consistency while offering straightforward, powerful tracking features without any need for an internet connection.

## Project Overview

Many people who are starting to work out do it on paper or mobile notes, which later gets mislead or recorded without consistency.

The Personal Fitness Tracker offers an offline, easy-to-operate solution for fitness monitoring, goal tracking, and progress visualization that will help to keep users motivated.

## Features
- Log daily workouts: exercise, duration, calories
- Set weekly goals calories + workout count
- View weekly summary
- Goal accomplishment checker
- Text-based progress visualization bar
- BMI calculator with health categories
- View full workout history
- Delete incorrect Entries
- Auto saves progress

## Technologies/Tools used
Programming Language - Python 3.x

Data Storage - JSON File

Environment - CLI (Terminal / Command Prompt)

## Steps to install and run the project
- Install Python
- Dowload the project
- Run the programme
- Start using the tracker

## Usage Instruction

1. Log Workout
 -  Enter exercise name, duration (minutes), and calories burned.

2. Set Goal
  - Set weekly calorie goal or number of workouts goal.

3. View Weekly Summary
  - Shows total workouts and total calories burned in the current week.

4. Check Goals
  - Displays whether weekly goals have been achieved or how much is remaining.

5. Visualize Progress
  - Shows a text-based progress bar (1 bar = 100 calories).

6. BMI Calculator
  - Enter weight (kg) and height (cm) to calculate BMI category.

7. View Workout History
  - Lists all stored workout entries with date and details.

8. Delete Workout Entry
  - Remove an incorrect entry using its serial number from history.

9. Save & Exit
  - Saves all progress in fitness_data.json and closes the program.

### Data Persistence
- All workout logs and goals are automatically saved locally.
- When the app is opened again, previous data will be reloaded.
